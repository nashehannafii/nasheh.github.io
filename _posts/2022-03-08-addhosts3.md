Saya menggunakan ubuntu linux.

## 1. Instal Node Version Manager

```bash
npm install -g n
```

## 2. Instal Versi Node Spesifik

```bash
# instal versi spesifik
n 16.4.0

# instal versi terbaru
n latest

# instal versi lts
n lts 
```

Terkadang membutuhkan sudo.
